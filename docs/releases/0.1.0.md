## 新功能

forgectl 首個版本, 透過 GitHub / GitLab REST API 查詢與操作 repo 的 release 與 asset:

- 跨平台支援 GitHub / GitLab, 涵蓋公開站與自架站.
- `ping`: 驗證連線與認證設定.
- `release list`: 列出 repo 的所有 release.
- `release create`: 發布某 version 的正式 release.
- `asset upload`: 將本地檔上傳為某 version 的 asset.
- `asset download`: 下載某 release 的 asset.
- 以 credential 檔 (TOML / YAML / JSON) 作為認證的單一事實來源.
- 提供 `--insecure` 旗標, 對自架 host 跳過 TLS 憑證驗證.
